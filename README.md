# pangu

Birdthings pangu project

## Usage

1. Add dependences.

    $ lein deps

2. Compile ClojureScript.

    $ lein cljsbuild auto

3. Compile sass

    $ compass watch

3. Run Server.

    $ lein run

## Note

1. Make sure you run $ compass clean and $ lein clean every times when you do git commit.
